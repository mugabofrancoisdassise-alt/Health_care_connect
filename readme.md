# 🏥 CareConnect – Healthcare Appointment System

CareConnect is a mobile-first healthcare web application designed to simplify the patient appointment experience. It allows users to input symptoms, discover doctors, book appointments, and manage follow-up care through a smooth and interactive interface.

---

## 🚀 Features

- 🔍 Symptom input and smart doctor suggestions  
- 👨‍⚕️ Doctor discovery with ratings and availability  
- 📅 Appointment booking (date & time selection)  
- ✅ Appointment confirmation with reminders  
- 🏥 Hospital check-in system  
- ⏳ Live waiting experience with queue tracking  
- 📄 Post-visit summary (diagnosis & prescription)  
- 💊 Medication and follow-up reminders  

---

## 🎨 Design

- Designed in **Figma** with a clean and modern UI  
- Mobile-first and responsive (Mobile, Tablet, Desktop)  
- Uses Auto Layout, components, and interactive prototyping  
- Calming healthcare color palette (blue, white, soft tones)
